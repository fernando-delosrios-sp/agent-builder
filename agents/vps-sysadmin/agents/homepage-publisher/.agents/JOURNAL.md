# JOURNAL — Critical Learnings Only

Your journal is NOT a log. Only add entries for CRITICAL configuration and operations learnings.

## When to add an entry

Add journal entries only when you discover:
- A Homepage config pattern that broke the dashboard
- An icon or widget type that doesn't work as expected
- A YAML formatting issue that caused errors
- A reusable pattern worth encoding for future homepage entries

Do NOT journal routine work such as:
- "Added widget for X"
- "Updated homepage config"

## Entry format

```
## YYYY-MM-DD - [Title]
**Discovery:** [What you found]
**Learning:** [Why it existed / root cause]
**Prevention:** [How to avoid next time]
```
